# 浏览器概述及其内核

Q：什么是浏览器呢？

浏览器是安装在电脑里的一款软件，能够将网页内容呈现给用户查看，并让用户与网页交互的一种软件。

Q：常见的主流浏览器有哪些，及其内核？

|        浏览器        |                  内核                  |
| :------------------: | :------------------------------------: |
|          IE          |                Trident                 |
|       Firefox        |                 Gecko                  |
|    Google Chrome     |             WebKit-->Blink             |
|        Safari        |                 WebKit                 |
|        Opera         |       Presto--> Webkit --> Blink       |
| 360浏览器/猎豹浏览器 |            IE+Chrome双内核             |
| 搜狗、遨游、QQ浏览器 | Trident（兼容模式）+Webkit（高速模式） |
| 百度浏览器、世界之窗 |                 IE内核                 |
|      2345浏览器      |       IE内核 --> IE+Chrome双内核       |
